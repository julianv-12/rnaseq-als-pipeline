# RNA-seq ALS Pipeline

## Overview
End-to-end bulk RNA-seq workflow for public ALS transcriptomic data, from raw sequencing files to differential expression and biological interpretation.

## Objectives
- Download public RNA-seq data
- Curate metadata
- Perform quality control
- Quantify transcript abundance
- Generate gene-level counts
- Perform exploratory analysis
- Identify differentially expressed genes
- Interpret results with pathway enrichment
- Build a reproducible pipeline with Nextflow and Docker
